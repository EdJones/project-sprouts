Overview

The FDB task provides an interface to the Flash Debugger.

In order to use this tool, you'll need to compile a SWF file with --debug=true, and be prepared to open it in a debug Flash Player. You can open the SWF using the desktop debug Flash Player for your platform using the FlashSDK::FlashPlayer task, or you can open the SWF manually on the desktop or the browser - as long as you run it in a Debug Flash Player.
